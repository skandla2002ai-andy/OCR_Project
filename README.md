# OCR Project

OCR (Optical Character Recognition) 프로젝트입니다.

## 프로젝트 소개

이 프로젝트는 이미지에서 텍스트를 추출하는 OCR 기능을 제공합니다.

## 설치 방법

```bash
# 필요한 패키지 설치
pip install -r requirements.txt
```

## 사용 방법

```bash
# 프로젝트 실행
python main.py
```

## 주요 기능

- 이미지에서 텍스트 추출
- 다양한 이미지 형식 지원
- 정확한 문자 인식

## 기술 스택

- Python
- OCR 라이브러리
